# LA OPERATIVA

Sitio web oficial preparado para GitHub Pages y TikTok Developers.

## Características

- HTML5
- CSS3
- JavaScript
- Responsive
- SEO
- PWA
- GitHub Pages

## Publicación

1. Sube el proyecto a GitHub.
2. Ve a Settings > Pages.
3. Selecciona:
   - Deploy from a branch
   - main
   - /(root)
4. Guarda los cambios.
